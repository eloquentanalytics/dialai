# Installation

Install DialAI using npm:

```bash
npm install dialai
```

Or with yarn:

```bash
yarn add dialai
```

Or with pnpm:

```bash
pnpm add dialai
```

## Requirements

- Node.js 18.0.0 or higher
- TypeScript 5.0.0 or higher (recommended)

## Next Steps

Once installed, check out the [Quick Start](/docs/getting-started/quick-start) guide to create your first session.
