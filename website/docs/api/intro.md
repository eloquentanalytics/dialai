# API Reference

The DialAI API provides functions for creating sessions, registering specialists, and managing the decision cycle.

## Core Functions

### `createSession(options)`

Creates a new session instance.

### `registerSpecialist(options)`

Registers a specialist for a session type.

### `submitProposal(options)`

Submits a proposal for a session.

### `submitVote(options)`

Submits a vote comparing two proposals.

### `executeTransition(options)`

Executes a transition in a session.

### `evaluateConsensus(options)`

Evaluates consensus using the registered Arbiter strategy.

## Types

All TypeScript types are exported from the main package. See the generated TypeDoc documentation for detailed type information.
